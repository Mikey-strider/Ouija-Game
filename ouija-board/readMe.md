# Ouija Board!

I am trying to build a fun, unique game, where you need to solve riddles and puzzles on a web browser. I tend to gravitate towards paranormal themes, which is why I have chosen an ouija board as my first online puzzle theme.

## MVP

The game will present 3 riddles in which to solve. Once third riddle is solved the game will change. You will then be presented with 6 tarot cards you will have align just right in order to find a hidden message.
